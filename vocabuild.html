OVERVIEW

Vocabuild is a personal project for a UX course which I undertaken with CareerFoundry. The aim of this app is to empower people to learn new vocabulary. People of all backgrounds are encouraged to improve their language skills and increase communication level. This application is meant to be used for quick 5–10 minute study sessions. That could be on a bus or train, as well as waiting between classes, on a lunch break, in a cafe or in a bed etc.

MY ROLE

The entire design process including Research, Competitive Analysis, User Surveys and Interviews, User Personas, User Flows, Wireframes, Usability Testing, Affinity Mapping, UI Design and High Fidelity Prototype.

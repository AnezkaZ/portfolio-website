Perfect Properties

OVERVIEW

Perfect Properties is a responsive web app which I designed for a UI for UX designers course at CareerFoundry. This app provides to new small-scale property buyers with expertise needed to get started efficiently. A database of available residential properties as well as  comprehensive information on each listing is presented with a user-friendly manner. Buyers can use this tool at home or on the go as long as they are logged in on a device.

MY ROLE

User Flows, Wireframes, Usability Testing, Affinity Mapping, UI Design and High Fidelity Prototype.

TOOLS

Paper and pencil, Balsamiq, Sketch, InDesign, Illustrator, Photoshop, After Effects, InVision.
